Iterative Statements / Loops ->
		1) for Loop
			syntax ->       1               2		     4
				for(initialization; condition; increment/decrement/expression){
					      3
					//code or body 
				}

		2) while loop
			
			syntax ->
				initialization
				while(condition){
					//code or body

					increment/decrement/expression
				}

		3) do-while loop -> It Execute at least Once.
			
			syntax ->
				do{
					//code or body
				}while(condition);
